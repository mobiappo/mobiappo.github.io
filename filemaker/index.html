FileMaker

FileMaker on ohjelmistoalusta, jonka avulla voidaan kehittää ja käyttää omaan tarpeeseen räätälöityjä sovelluksia Windows- ja Mac-, iOS- ja WWW-ympäristöissä.

FileMaker Pro Advanced

Paras työkalu niin FileMaker-sovellusten käyttöön kuin kehittämiseen. FileMaker Pro Advancedilla saa parhaan käyttökokemuksen FileMaker-sovelluksiin.

FileMaker WebDirect

Joustava tapa käyttää FileMaker-sovelluksia ilman asennuksia. FileMaker Serveriltä hostattua sovellusta voi käyttää lähes millä tahansa päätelaitteella. WebDirectin avulla FileMaker sovelluksia voi käyttää myös Androidissa ja Linuxissa.

FileMaker Go

FileMaker Go -sovellukset kehitetään FileMaker Pro:lla, mutta niitä käytetään Go:lla. Nopein tapa kehittää mobiilisovellusta liiketoiminnan tarpeisiin. Toimii iPadilla ja iPhonella.

Integraatiot

FileMaker-sovellukset keskustelevat sujuvasti muiden järjestelmien kanssa. Olemme toteuttaneet integraatioita esimerkiksi kirjanpito-ohjelmiin, laskutusohjelmiin, postin kuljetusasiakirjojen tilauskanaviin, massasähköpostipalveluihin, tekstiviestipalveluihin, Slackiin sekä Googlen GSuite -palveluihin.

Hostaus

FileMaker Server 16 ja FileMaker Server 17 -hostaus AWS:ssä (Amazon Web Services) luotettavien ja nopeiden yhteyksien päässä. Käytämme Windows Server 2012 ja 2016 palvelimia, jotka ovat dedikoitu vain ja ainoastaan FileMaker Serverin käyttöön. Hostauksen hinta riippuu käytön määrästä, käyttäjien määrästä sekä käyttötavoista. 5-10 käyttäjän tyypilliseen käyttöön hinta alkaen 150 e/kk (+alv). Teemme aina asiakaskohtaisen ratkaisun, jotta palvelin soveltuu tarpeisiinne mahdollisimman hyvin.
